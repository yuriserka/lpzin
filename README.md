## Instruções de uso

Execute:

> `$cd ./front`
>
> `$yarn`
>
> `$yarn build`

Para rodar o servidor precisa abrir outro terminal na pasta raíz e executar:

> `$go build`
>
> ./lpzin

A aplicação irá escutará na porta 8080.
